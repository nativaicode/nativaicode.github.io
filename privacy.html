Kebijakan Privasi

Dokumen ini menjelaskan bagaimana aplikasi menggunakan izin Google Drive dan melindungi data pengguna.

Ringkasan:
Aplikasi hanya meminta akses Google Drive untuk membuat, membaca, dan mengelola file yang dibuat oleh aplikasi sendiri. Data Anda tidak pernah dikirim ke server kami.

Penggunaan Google Drive

Aplikasi menggunakan izin Google Drive hanya untuk:

  • Membuat file yang diperlukan oleh aplikasi.
  • Membaca file yang sebelumnya dibuat oleh aplikasi.
  • Memperbarui atau menghapus file milik aplikasi jika diperlukan.

Aplikasi tidak mengakses file lain di Google Drive Anda yang bukan dibuat oleh aplikasi.

Penggunaan Data

Semua data diproses langsung antara aplikasi dan akun Google Drive Anda melalui Google Drive API. Kami tidak mengumpulkan, menyimpan, atau mengirimkan data tersebut ke server milik kami.

Keamanan

Aplikasi menggunakan mekanisme autentikasi resmi dari Google (OAuth) untuk memastikan akses hanya diberikan setelah Anda memberikan izin.

Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui apabila terdapat perubahan pada fitur atau penggunaan layanan. Versi terbaru akan selalu tersedia pada halaman ini.
